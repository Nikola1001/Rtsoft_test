Тестовое задание.
Целью данного "проекта" является простота. Он не продразумевает большие изменения. 
Иначе имел бы смыл реализовывать на Django. 
Проект реализован на фреймворке Flask только с базовыми библиотеками, за исключением библиотеки csv.
Считывание файла конфигурации происходит при запуске сервера. 
Это плохо тем, что при сбое в работе сервера и его перезапуске
файл будет считываться снова, но о других условиях в задании не сказано.
Сначала выводятся картинки с наибольшим числом показов для соблюдения баланса. 
Тесты написаны только чтобы убедиться, что сервер работает.
